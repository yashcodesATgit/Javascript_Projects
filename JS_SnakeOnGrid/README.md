# JS Snake Game 🐍

A modern, neon-themed Snake game built with pure HTML, CSS, and JavaScript. No frameworks, no libraries — just clean beginner-friendly code.

## Live Preview

Open `index.html` directly in your browser, or serve it locally with any static server.

## Features

- 25×25 circular neon grid
- Glowing white snake and red food
- Score, High Score (saved in localStorage), and Timer
- Keyboard arrow key controls
- On-screen D-pad for mobile
- Pause button
- Game Over screen with Restart

## How to Play

1. Open `index.html` in your browser
2. Use **Arrow Keys** or the on-screen buttons to move the snake
3. Eat the red dot to grow and score points
4. Avoid hitting the walls or your own body
5. Press **❚❚** to pause at any time

## Controls

| Action | Key |
|---|---|
| Move Up | Arrow Up |
| Move Down | Arrow Down |
| Move Left | Arrow Left |
| Move Right | Arrow Right |
| Pause | ❚❚ button |
