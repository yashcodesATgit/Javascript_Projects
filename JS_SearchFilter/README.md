# 🔍Javascript based Search Filter

A modern, responsive candidate directory and real-time search application built using clean, vanilla **HTML5, CSS3, and JavaScript**. 

The app features a glassmorphic design language, smooth layout animations, and optimized debounced searching for finding team members and skills instantly.

---

## 🚀 Key Features

- **Real-Time Filtering**: Dynamic, case-insensitive keyword filtering across candidate names and roles.
- **Performance Optimized**: Debounced search event handling to prevent unnecessary layout recalculations.
- **Premium Glassmorphic UI**: Deep slate radial gradient backgrounds, fine frosted borders, and high-contrast readable text overlays.
- **Responsive Layout**: Designed with a flexible CSS Grid that scales seamlessly across mobile, tablet, and desktop screens.
- **Micro-Animations**: Hover zoom transitions on card backgrounds and glowing active borders on the search input.
- **Bug-Free Design**: Clean rendering methods with built-in empty search state handlers and error-free JS runtime execution.

---

## 🛠️ Tech Stack

- **HTML5**: Semantic markup structuring.
- **CSS3**: Custom grid systems, gradients, transitions, and inline vector graphic integrations.
- **JavaScript (ES6+)**: Custom DOM manipulation, input debouncing, and list filtering logic.
