# ⏳ Pomodoro Timer

A clean and responsive Pomodoro Timer built using **HTML, CSS, and JavaScript**.  
The project helps users focus on productivity using the Pomodoro technique.

---

## 🚀 Features

- ✅ Start Timer
- ✅ Pause Timer
- ✅ Reset Timer
- ✅ Dynamic countdown
- ✅ Responsive UI
- ✅ Modern gradient design
- ✅ Real-time DOM updates

---

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript
